# INTERN1:End-to-end Omnichannel Retail Sales & Inventory Analytics Dashboard — 
data cleaning pipeline, SQL aggregations, and interactive BI visualizations 
built with React, Three.js, and Recharts.

## 📌 Project Overview

This project is an end-to-end Data Analytics solution developed as part of 
a 4-week internship program. It addresses a critical business problem faced 
by multi-location retailers transitioning from offline to digital models — 
the inability to unify fragmented sales data across brick-and-mortar POS 
systems and online storefronts.

The platform ingests raw retail transaction CSV data, runs an automated 
data cleaning and transformation pipeline, computes key business metrics 
through SQL-style aggregations, and presents actionable intelligence through 
an interactive multi-page Business Intelligence dashboard.

The solution serves three primary user personas:
- Store Managers tracking daily localized sales performance
- Inventory Planners forecasting seasonal demand and minimizing dead stock
- Regional Sales Directors analyzing geographic revenue distribution and ROI

Abstract

This report presents the design and implementation of an Omnichannel Retail 
Sales and Inventory Analytics Dashboard developed during a 4-week data 
analytics internship. The project addresses the operational challenge faced 
by multi-location retail enterprises in consolidating transactional data 
from disparate Point-of-Sale (POS) systems and digital storefronts into a 
unified analytical platform.

The solution encompasses four key components: an automated data cleaning 
and transformation pipeline that processes raw CSV transaction logs; a 
SQL-based aggregation layer that extracts critical business metrics including 
Total Revenue, Average Revenue per Order, and Geographic Sales Distribution; 
an interactive six-page Business Intelligence dashboard featuring dynamic 
time-series visualizations, product performance analysis, inventory 
forecasting, and three-dimensional data models; and an auto-generated 
insights report providing strategic business recommendations.

The platform was validated against three user personas — Store Managers, 
Inventory Planners, and Regional Sales Directors — and successfully 
demonstrates the ability to filter data across timeframes, locations, and 
product categories to enable data-driven decision-making.

Keywords: Data Analytics, Business Intelligence, Retail Analytics, 
Data Cleaning, Dashboard, Inventory Forecasting, React, Three.js
